Bitcoin Private Key Finder v2.0

📝 Descrição
Um programa otimizado em Python para busca de chaves privadas Bitcoin usando processamento híbrido (GPU + CPU) com suporte específico para GPUs AMD.

🚀 Características
Processamento híbrido (GPU + CPU)
Otimizado para GPUs AMD
Sistema de checkpoint automático
Monitoramento em tempo real
Notificação via WhatsApp quando encontrar a chave
Gerenciamento inteligente de memória
Suporte a múltiplos núcleos de CPU
Interface de linha de comando com estatísticas detalhadas

📋 Pré-requisitos
- Python 3.8+
- GPU AMD com suporte OpenCL
- Drivers AMD atualizados
- WhatsApp Web configurado (para notificações)
